# TECBOOK - Tu Biblioteca Digital



##Descripción

*TECBOOK* es una plataforma de ebooks estilo Netflix que permite leer libros de dominio público con una experiencia de usuario moderna y atractiva. Guarda automáticamente tu progreso de lectura y ofrece múltiples temas visuales.

# Características Principales

-  **Interfaz tipo Netflix** con diseño moderno y atractivo
-  **Lector integrado** con navegación entre páginas
-  **Guardado automático** del progreso de lectura
-  **Múltiples temas**: claro, oscuro y sepia
-  **Ajuste de tamaño de letra** para mejor lectura
-  **Vistas organizadas**: Inicio, Mi Biblioteca, Continuar leyendo
-  **Estadísticas** de lectura
-  **Totalmente legal** (solo libros de dominio público)
